取得 GitHub Token 步驟
1. 開啟 GitHub → Settings → Developer settings → Personal access tokens → Tokens (classic)
2. 點「Generate new token (classic)」
3. Note 隨意填，Expiration 選你要的期限
4. 勾選 repo（整個 repo 權限）
5. 點「Generate token」，複製 ghp_ 開頭的 Token
6. 貼到上方欄位並儲存
